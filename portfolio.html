Predictive Game Analysis Program

I developed a Python-based predictive analytics program that integrates SQL queries and linear regression models to analyze and forecast Luka Dončić’s past and future game performances. Using historical game data from a structured SQL database, I built multiple linear regression models to identify key performance trends and make data-driven predictions. Some key aspects of my project include predictive modeling, statistical analysis, and graphical data visualization. This project allowed me to combine data science, SQL, and machine learning to extract meaningful insights from basketball data while sharpening my skills in data-driven decision-making and predictive modeling.

y86-64 Processor

I designed and implemented a fully functional Y86-64 processor capable of executing Y86-64 assembly code according to the Y86-64 Instruction Set Architecture (ISA). The processor was built with a 5-stage pipelining architecture, incorporating the stages of Fetch, Decode, Execute, Memory Access, Writeback, and PC Update. The goal of the project was to develop an efficient processor pipeline that ensured correct instruction execution while optimizing throughput and handling potential hazards.

Pharmaceutical Price Regulation Research Paper

This research paper examines the unintended consequences of pharmaceutical price regulation, focusing on market shifts, consumer access, and innovation. Using regression analysis on sales data before and after price caps in India, the study finds that regulation reduces the availability of essential medicines and discourages innovation as firms shift focus to non-capped drugs. It highlights key marketing concepts like pricing strategy, consumer behavior, and market competition, emphasizing the need for policymakers to consider industry responses when designing regulations.
